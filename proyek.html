Extension Buka Banyak Tab

Automation Tool (Batch & Excel Integration)

Tools: Notepad (Batch scripting), Microsoft Excel, VSCode

Proyek pribadi yang saya buat saat menjabat sebagai Image Editor. Extension berbentuk file batch (.bat) yang memungkinkan membuka banyak tab/URL atau folder sekaligus hanya dengan dua kali klik. File ini bisa diedit langsung di Notepad untuk menambah/mengurangi daftar link.

Alur Kerja Pembuatan

  1. Identifikasi masalah: membuka puluhan tab produk/gambar secara manual memakan waktu lama.
  2. Buat script batch sederhana (.bat) dengan perintah start untuk membuka URL/folder.
  3. Integrasi dengan Excel: list URL di sheet, export ke .bat via macro sederhana.
  4. Test & debug: pastikan kompatibel Windows, handle error jika link invalid.
  5. Distribusi: bagikan ke tim sebagai file executable sederhana.

Case Study Mini

Tujuan: Menghemat waktu tim saat menangani batch editing produk di e-commerce.
Tantangan: Proses manual repetitif, rawan salah klik, dan memakan waktu hingga 5–10 menit per sesi.
Solusi: File .bat yang bisa diedit Notepad + opsi list dari Excel untuk fleksibilitas.
Hasil: Mengurangi waktu buka tab hingga 70–80%, digunakan oleh beberapa rekan editor.

Extension Auto Click

Browser / VSCode Automation Tool

Tools: JavaScript, VSCode, Browser Extension (Manifest V3)

Proyek pribadi saat menjabat sebagai Rich Content Specialist. Extension yang di-install di browser (atau VSCode) untuk otomatis mengklik tombol/elemen tertentu pada halaman/tab yang sama dalam satu window. Sangat berguna untuk tugas repetitif seperti approve konten atau klik tombol di dashboard.

Alur Kerja Pembuatan

  1. Identifikasi tugas repetitif di dashboard e-commerce (klik tombol berulang).
  2. Buat extension sederhana dengan content script JS untuk detect elemen target.
  3. Tambahkan popup/button trigger untuk mengaktifkan auto-click.
  4. Test di browser (Chrome/Edge) dan VSCode jika diadaptasi.
  5. Debug: pastikan hanya berjalan di tab/window aktif, tidak mengganggu user experience.

Case Study Mini

Tujuan: Mengotomatiskan klik berulang untuk meningkatkan produktivitas tim content.
Tantangan: Tugas klik manual di dashboard memakan waktu dan rentan human error.
Solusi: Extension yang trigger klik pada elemen spesifik saat diaktifkan, aman di satu tab/window.
Hasil: Mengurangi waktu task hingga 40–60%, meningkatkan efisiensi harian.

Website ElektroArt

Website Portal Teknologi (Tugas Kampus)

Tools: Visual Studio Code, HTML5, CSS3, JavaScript

Website portal teknologi bernama ElektroArt yang berisi informasi produk elektronik (laptop, smartphone, tablet), artikel tutorial, dan berita teknologi. Dirancang sebagai media informasi yang informatif dan menarik secara visual.

Alur Kerja Pembuatan

  1. Riset konten: kategori produk elektronik + topik tutorial/berita.
  2. Desain struktur: homepage → kategori produk → artikel detail.
  3. Implementasi: HTML untuk struktur, CSS responsif, JS untuk interaksi sederhana (filter, carousel).
  4. Optimasi visual: layout grid, card produk, dark/light mode toggle.
  5. Test & deploy: cross-browser, mobile responsive.

Case Study Mini

Tujuan: Membuat portal informasi teknologi yang user-friendly dan menarik untuk pembaca umum.
Tantangan: Mengemas informasi teknis agar mudah dipahami tanpa mengurangi kedalaman.
Solusi: Gunakan card visual + gambar produk + artikel ringkas dengan navigasi mudah.
Hasil: Proyek tugas kampus mendapat nilai baik, menjadi dasar belajar front-end lebih lanjut.

Aplikasi - Tampilan Scan

Aplikasi Barcode Scanner Parking

Mobile App (Tugas Kampus)

Tools: Android Studio, Kotlin + Jetpack Compose, Google Sheets API

Sistem informasi manajemen parkir berbasis Android untuk kawasan industri. Menggunakan pemindaian barcode pada stiker kendaraan sebagai kontrol akses masuk/keluar.

Komplek industri merupakan kawasan dengan aktivitas tinggi... (abstrak lengkap seperti yang kamu berikan, saya singkatkan di sini untuk tampilan rapi)

Alur Kerja Sistem

  1. Karyawan ajukan stiker parkir ke personalia.
  2. Personalia verifikasi dokumen & lanjut ke GA.
  3. GA terbitkan stiker dengan barcode unik.
  4. Petugas keamanan scan barcode di gerbang untuk log masuk/keluar.
  5. Data tersimpan di Google Sheets untuk monitoring & laporan.

Case Study Mini

Tujuan: Menciptakan tata kelola parkir yang tertib, aman, dan terkontrol di kawasan industri.
Tantangan: Parkir sembarangan, melebihi kapasitas, dan sulit monitoring manual.
Solusi: App Android + scan barcode + integrasi Sheets untuk log real-time.
Hasil: Proyek tugas kampus berhasil, area parkir lebih tertata, keamanan meningkat.

Sistem Manajemen Visitor (Flutter + PHP)

Hybrid Web & Mobile App (Tugas Kampus)

Tools: VSCode, Flutter (mobile), PHP + MySQL (backend), XAMPP, HTML/CSS/JS (web)

Aplikasi manajemen visitor untuk kompleks industri ADR Group: registrasi mandiri, validasi QR multi-pos, monitoring pergerakan, dan pelaporan data.

Penelitian ini membahas perancangan dan pengembangan aplikasi manajemen visitor... (abstrak lengkap, saya ringkas untuk tampilan)

Alur Kerja Sistem

  1. Tamu registrasi mandiri via web (Bootstrap/React).
  2. Admin verifikasi & generate QR Code.
  3. Visitor berjalan ke pos tujuan.
  4. Security di pos tujuan validasi menggunakan app Flutter (scan QR).
  5. Visitor selesai → kembali ke pos utama untuk validasi akhir & akhiri kunjungan.

Case Study Mini

Tujuan: Mendigitalisasi proses kunjungan tamu untuk efisiensi dan keamanan lebih tinggi.
Tantangan: Sistem manual rentan kesalahan, keterlambatan laporan, dan sulit tracking real-time.
Solusi: Hybrid app: web untuk registrasi/admin, mobile Flutter untuk scan QR multi-pos.
Hasil: Proses verifikasi lebih cepat, log aktivitas lengkap, meningkatkan kontrol keamanan.
